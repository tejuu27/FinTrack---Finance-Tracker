# 💰 FinTrack - Finance Tracker

A simple personal finance tracker built with React + Vite.

It helps users track income, expenses, budgets, and analyze spending in a clean dashboard UI.

---

## 🚀 Features

- Add income & expense transactions
- Delete & manage transactions
- Budget tracking system
- Persistent storage using LocalStorage
- Clean dashboard overview
- Category-based organization

---

## 🛠️ Tech Stack

- React.js
- Vite
- Context API
- JavaScript (ES6+)
- CSS

---

## 📊 Pages

- Dashboard – Overview of finances
- Transactions – Manage all entries
- Add Transaction – Add income/expense
- Budget – Set monthly budget
- Analytics – Spending insights (basic)

---
